Demostración del AI Chat Widget

Esta página muestra cómo integrar el widget de chat con IA en cualquier sitio web utilizando un único script.

Cómo funciona

El widget extrae automáticamente los encabezados (h1, h2) de esta página para proporcionar contexto a la IA. Prueba a preguntar sobre nuestros servicios o características del widget.

Para integrar este widget en tu propio sitio web, simplemente añade el siguiente código:

<script 
  src="/assets/ai-chat-widget.js" 
  data-chat-widget 
  data-auto-init="true" 
  data-api-url="https://chatbot-widget-code-base.onrender.com/api/chat"
  data-system-prompt="Eres un asistente amigable que ayuda a los usuarios con preguntas sobre nuestra empresa.">
</script>

Puedes interactuar con el widget haciendo clic en el icono de chat en la esquina inferior derecha de la pantalla.

Personalización

El widget puede personalizarse completamente para adaptarse a las necesidades de tu sitio web, incluyendo:

  • Cambiar los colores y estilos
  • Modificar el prompt del sistema
  • Ajustar el comportamiento de la IA
