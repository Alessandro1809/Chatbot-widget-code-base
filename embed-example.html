Integración del Widget de Chat con IA

Esta página muestra cómo integrar el widget de chat con IA en cualquier sitio web utilizando un único script.

Opción 1: Auto-inicialización

<script 
  src="https://tu-dominio.com/assets/ai-chat-widget.js" 
  data-chat-widget 
  data-auto-init="true" 
  data-api-url="https://tu-backend.com/api/chat"
  data-system-prompt="Tu prompt personalizado aquí">
</script>

Opción 2: Inicialización manual

<script src="https://tu-dominio.com/assets/ai-chat-widget.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    window.AIChatWidget.init({
      apiUrl: 'https://tu-backend.com/api/chat',
      systemPrompt: 'Tu prompt personalizado aquí'
    });
  });
</script>

Opciones de Configuración

  • data-api-url: URL del endpoint de la API del backend
  • data-system-prompt: Prompt personalizado del sistema para la IA
  • data-auto-init: Si se establece en "true", el widget se inicializará automáticamente

Personalización del Prompt

El prompt del sistema permite personalizar el comportamiento de la IA según las necesidades específicas de cada sitio web. Por ejemplo:

Eres un asistente virtual para una tienda de ropa. Tu objetivo es ayudar a los usuarios a encontrar productos, responder preguntas sobre tallas, materiales y políticas de devolución. Debes ser amable y profesional en todo momento.

Extracción de Contexto

El widget extrae automáticamente los encabezados (h1, h2) de la página para proporcionar contexto a la IA. Esto permite que las respuestas sean relevantes al contenido de la página.
